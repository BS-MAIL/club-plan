# 5. 개발 단계

## 1단계: 기본 프로젝트 만들기

- `jkh919` 폴더 안에 파이썬 웹 프로젝트 구조 만들기
- Flask 설치
- 기본 메인 페이지 만들기

## 2단계: 입력 화면 만들기

- 부등식 두 개를 입력받는 HTML 폼 만들기
- 입력값을 서버로 전달하기

## 3단계: 수학 계산 기능 만들기

- 입력된 이차부등식을 파싱하기
- 각 부등식의 해 구간 구하기
- 두 해 구간의 교집합 구하기

## 4단계: 그래프 기능 만들기

- 각 이차함수 그래프 그리기
- 해가 되는 구간을 시각적으로 표시하기
- 공통 해 구간을 강조하기

## 5단계: 설명 기능 만들기

- 계산 결과를 바탕으로 단계별 설명 문장 만들기
- 어려운 수학 용어는 최대한 쉽게 풀어쓰기

## 6단계: 예제와 연습 문제 추가

- 예제 문제 버튼 만들기
- 난이도별 문제 추가
- 정답 확인 기능 추가

## 7단계: 디자인 개선

- 보기 쉬운 색상 사용
- 입력창과 결과 영역 정리
- 모바일 화면에서도 보기 좋게 수정

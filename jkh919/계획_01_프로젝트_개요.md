# 1. 프로젝트 개요

## 프로젝트 목표

연립이차부등식을 어려워하는 사용자가 그래프와 단계별 설명을 통해 개념을 쉽게 이해할 수 있는 파이썬 기반 웹 애플리케이션을 만든다.

사용자는 이차부등식 두 개를 입력하고, 웹은 다음 내용을 보여준다.

- 각 부등식의 그래프
- 각 부등식이 만족되는 영역
- 두 부등식을 동시에 만족하는 공통 해 영역
- 풀이 과정에 대한 쉬운 설명
- 예제 문제와 연습 문제

## 예상 사용자

- 중학생 또는 고등학생
- 연립이차부등식을 처음 배우는 학생
- 그래프로 부등식의 의미를 이해하고 싶은 사람
- 수학 문제 풀이 과정을 단계별로 확인하고 싶은 사람

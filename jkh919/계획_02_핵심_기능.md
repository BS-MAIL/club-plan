# 2. 핵심 기능

## 2.1 부등식 입력 기능

사용자가 두 개의 이차부등식을 입력할 수 있게 한다.

예시:

```text
x^2 - 4x + 3 > 0
x^2 - 1 <= 0
```

처음에는 입력 형식을 너무 자유롭게 만들기보다, 정해진 형식부터 지원한다.

예시 입력 방식:

- `ax^2 + bx + c > 0`
- `ax^2 + bx + c >= 0`
- `ax^2 + bx + c < 0`
- `ax^2 + bx + c <= 0`

## 2.2 그래프 시각화 기능

각 이차함수의 그래프를 화면에 보여준다.

표시할 내용:

- 포물선 그래프
- x축과 만나는 점
- 꼭짓점
- 부등식을 만족하는 x 구간
- 두 부등식의 공통 해 구간

## 2.3 풀이 과정 설명 기능

사용자가 단순히 정답만 보는 것이 아니라, 왜 그런 답이 나오는지 이해할 수 있게 한다.

설명 흐름:

1. 각 이차식의 근을 구한다.
2. 포물선의 방향을 확인한다.
3. 부등호에 따라 참이 되는 구간을 찾는다.
4. 두 부등식의 해 구간을 비교한다.
5. 공통으로 겹치는 구간을 최종 답으로 제시한다.

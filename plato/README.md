# PLATO: 이차함수 그래프 탐구 웹앱 기획

> **프로젝트명**: PLATO — Parabola Learning And Thinking Observatory  
> **수업 주제**: 이차함수의 그래프를 이용한 탐구형 실습 웹앱  
> **핵심 질문**: “그래프를 직접 움직이면 이차함수의 식과 성질을 더 잘 이해할 수 있을까?”  
> **대상**: 중3~고1 수학 학습자  
> **활용 맥락**: MAIL 동아리 2차시 수업 산출물 A/B/C 제작을 위한 기획 문서

---

## 1. 기획 배경

이차함수 단원에서 학생들은 보통 `y = ax² + bx + c`, `y = a(x-p)² + q`의 식을 외우고 그래프의 모양을 문제 풀이에 적용합니다. 그러나 실제 수업에서는 다음과 같은 어려움이 자주 나타납니다.

- `a`, `p`, `q`, `b`, `c`가 그래프에 어떤 영향을 주는지 직관적으로 연결하지 못함
- 꼭짓점, 축의 방정식, y절편, x절편을 공식으로만 기억함
- 그래프의 이동·대칭·폭 변화를 동적인 과정으로 이해하지 못함
- “이차함수 그래프가 현실 문제와 무슨 관련이 있는가?”에 대한 흥미 부족

PLATO는 학생이 슬라이더와 미션을 통해 포물선을 직접 조작하고, 변화의 원인을 말로 설명하며, 게임형 과제를 해결하도록 설계하는 웹 기반 인터랙티브 교구입니다.

## 3. 앱의 핵심 콘셉트

### 한 문장 설명
학생이 포물선을 직접 조작하며 “식의 변화 ↔ 그래프의 변화 ↔ 수학적 의미”를 발견하는 이차함수 탐구 샌드박스입니다.

### 수업 속 역할
- **교사 설명 보조 도구**가 아니라, 학생이 직접 탐구하는 **디지털 조작형 수학 실험실**
- 문제 풀이 전 단계에서 개념을 시각화하는 **개념 형성 도구**
- 학생의 조작 기록과 설명을 수집하는 **과정 중심 평가 도구**

### 주요 활동
1. 슬라이더로 `a`, `p`, `q` 값을 바꾸며 그래프 변화 관찰
2. 미션 카드에 제시된 조건을 만족하는 포물선 만들기
3. 현실 맥락 과제: 농구공 궤적, 분수 물줄기, 다리 아치 등 모델링
4. 자신의 조작 과정을 설명하고 친구와 비교
5. 마지막에 “내가 발견한 이차함수 그래프 규칙” 정리

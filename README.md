# MAIL (Math-AI Interactive Lab) 동아리 활동 계획

> **부산광역시교육청 AI·디지털 교사 동아리**  
> **신청일**: 2026년 3월 27일  
> **대표**: 김규봉 (부산소프트웨어마이스터고등학교)

---

## 📌 동아리 소개

**MAIL (Math-AI Interactive Lab)**은 부산광역시교육청 디지털미래교육과 소속 AI·디지털 교사 연구 동아리입니다.

### 이름의 의미
- **M**ath: 수학교육의 본질 - 탐구와 발견
- **AI**: 인공지능 기술의 교육적 활용
- **I**nteractive: 학생-교구-데이터의 쌍방향 상호작용
- **Lab**: 실험하고 검증하는 연구 공동체

### 비전
> "AI는 도구이며, 교사가 설계한 진짜 탐구의 주인은 학생이다"

---

## 🎯 최종 목표

각 동아리원(교사)은 활동 기간 동안 아래 **3가지 핵심 산출물**을 완성합니다.

| 산출물 | 내용 | 형식 |
|:---:|:---|:---:|
| **A** | 2차시 수업 지도안 (상세안) | Markdown / Word |
| **B** | 실제 수업 수행 결과 | 영상/사진/학생작품/데이터 |
| **C** | 소감문 및 개선안 | Markdown / Word |

> 각 교사는 자신의 **담당 과목·학년·희망 주제**를 자율 선택하여 2차시 수업을 설계·수행·분석합니다.

---

## 📅 활동 기간

**2026년 3월 ~ 2027년 2월** (12개월)

### 월별 흐름

```
[3-5월]   기획·설계  → 주제 선정 → 목표 설정 → 지도안 초안
[6-8월]   개발       → 1차시 자료 → 2차시 자료 → 통합 테스트
[9-10월]  수행       → 1차시 현장 수업 → 2차시 현장 수업
[11-12월] 분석·정리  → 데이터 분석 → 개선안 → 최종 산출물
[1-2월]   확산·발표  → 보급 자료 → 성과 발표회 → 차년도 계획
```

## 📋 연구 주제

MAIL 동아리의 3가지 핵심 연구 영역입니다. 각 교사는 이 중 하나 이상을 자신의 수업에 적용합니다.

| 번호 | 연구 주제 | 상세 내용 |
|:---:|:---|:---|
| 가 | **디지털 조작형 수학 탐구 컴포넌트 설계** | 2022 개정 교육과정 기반, 학생이 변수를 직접 조작하며 원리를 발견하는 디지털 샌드박스(Sandbox) 환경 설계 |
| 나 | **OpenCode 기반 인터랙티브 교구 개발** | AI 코딩 비서(OpenCode)를 활용한 교사 제작 맞춤형 교구 개발 및 클라우드 배포 |
| 다 | **웹 기반 학습 데이터 로깅 시스템** | 이탈 행동, 문항별 체류 시간, 체감 난이도 등 다차원 학습 데이터 수집·분석 시스템 구현 |

> **참고**: 수학 외 과목 교사도 AI·디지털 도구를 활용한 탐구 중심 수업 설계가 가능합니다.

## 🛠️ 활동 원칙

1. **자율 선택**: 각 교사는 자신의 과목·학년·주제·AI 도구를 자율 선택
2. **협업 지원**: 개발 어려움 시 개발팀/콘텐츠팀과 1:1 매칭
3. **데이터 윤리**: 학생 개인정보 반드시 익명화, 동의서 필수 수집
4. **오픈 소스**: 개발 자료는 동아리 내 공유, 희망 시 외부 보급

---

## 📊 기대 효과

- **교사 주도적 디지털 혁신**: AI에 의존하는 수업이 아닌, 교사가 설계한 인터랙티브 교구로 수업 주도권 확보
- **수학적 사고의 시각화**: 학생들이 수학적 개념을 동적으로 조작하며 스스로 원리를 깨우치는 탐구 중심 수학교육 실현
- **데이터 기반 맞춤형 환류**: 학생의 탐구 궤적을 데이터로 읽어내고 AI가 분석함으로써 과정 중심 평가의 표준 모델 제시
- **오픈 소스 기반 확산**: mathbong.com 등 독자적 플랫폼을 토대로 한 교구 및 가이드를 부산교수학습샘터 및 디지털 교육 현장지원단 네트워크를 통해 보급

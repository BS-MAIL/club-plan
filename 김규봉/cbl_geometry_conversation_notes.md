## 2. CBL의 핵심 의미

CBL은 **Challenge-Based Learning**, 즉 **도전 중심 학습**이다.

일반적인 수업 흐름이

```text
개념 설명 → 예제 풀이 → 문제 풀이
```

이라면, CBL은

```text
현실 문제 → 핵심 질문 → 도전 과제 → 탐구 → 해결안 → 공유와 성찰
```

의 흐름을 가진다.

### CBL 기본 구조

| 단계 | 의미 |
|---|---|
| Big Idea | 사회적·생활적 의미가 큰 큰 주제 |
| Essential Question | 학생이 탐구할 핵심 질문 |
| Challenge | 실행 가능한 도전 과제 |
| Investigation | 조사, 탐구, 분석 |
| Solution/Action | 해결책 설계와 실행 |
| Reflection/Sharing | 성찰과 공유 |

## 3. PBL과 CBL의 차이

| 구분 | PBL | CBL |
|---|---|---|
| 출발점 | 교사가 제시한 문제 또는 프로젝트 | 큰 주제에서 학생이 구체적 도전 과제를 정의 |
| 핵심 질문 | “이 문제를 어떻게 해결할까?” | “이 현실 문제에 대해 우리가 무엇을 바꿀 수 있을까?” |
| 학생 역할 | 문제 해결자 | 문제 정의자이자 실행자 |
| 결과물 | 보고서, 발표, 산출물 | 실제 적용 가능한 제안, 행동, 개선안 |
| 평가 초점 | 문제 해결 과정과 결과 | 문제 정의, 탐구, 실행, 영향, 성찰 |

핵심 차이:

> PBL은 주어진 문제를 잘 해결하는 데 초점이 있고, CBL은 어떤 문제가 중요한지 정의하고 실제 맥락에서 변화를 시도하는 데 초점이 있다.

## 4. 강의 내용 기록

### 4.1 학생은 언제 몰입하는가?

- 강사는 수업 설계의 출발점으로 **“학생들이 언제 몰입하는가?”**를 질문함.
- 몰입은 정답을 들을 때보다, **자신이 해결해야 할 문제를 발견하거나 선택할 때** 발생함.

### 4.2 답을 아는 것의 가치 변화

- AI 시대에는 **답을 아는 것의 가치가 달라짐**.
- AI가 정보 검색, 요약, 계산, 초안 작성을 빠르게 수행함.
- 교육은 단순 정답 획득에서 벗어나야 함.

### 4.3 AI가 못하는 것: 문제 정의

- AI는 주어진 질문에 답하거나 자료를 정리하는 데 강함.
- 하지만 AI가 현실 맥락 속에서 의미 있는 문제를 발견하고 정의하는 것은 어려움.
- AI 시대 인간에게 중요한 역량은 **문제를 정의하는 능력**임.

핵심 문장:

> AI 시대에는 답을 아는 것보다, 어떤 문제를 정의할 것인가가 더 중요해졌다.

### 4.4 2022 개정 교육과정과 CBL

강사는 2022 개정 교육과정의 방향과 CBL이 연결된다고 설명함.

강조된 요소:

- 역량 중심 교육
- 깊이 있는 학습
- 질문 중심 교육
- 핵심 역량 강화
- 협업의 가치

정리 문장:

> 2022 개정 교육과정이 강조하는 역량 중심 교육과 깊이 있는 학습은 CBL의 문제 정의, 탐구, 협업, 성찰 과정과 밀접하게 연결된다.

### 4.5 AI 시대 학생에게 필요한 세 가지 힘

강사는 AI 시대 학생에게 필요한 핵심 역량을 다음 세 가지로 정리함.

```text
질문, 판단, 실행
```

| 힘 | 의미 | CBL과의 연결 |
|---|---|---|
| 질문 | 무엇이 중요한 문제인지 묻는 힘 | Essential Question |
| 판단 | 정보와 해결안의 타당성을 검토하는 힘 | Investigation, Solution |
| 실행 | 해결안을 실제로 표현·적용·공유하는 힘 | Action |

핵심 문장:

> AI가 답을 생성할 수 있는 시대일수록 학생은 무엇을 물을지, 무엇을 믿을지, 어떻게 실천할지를 배워야 한다.

### 4.6 강의 표어

강사가 제시한 핵심 표어:

> 좋은 질문은 탐구를 만들고,  
> 탐구는 문제 해결을 만들며,  
> 문제 해결은 도전과 실천으로 이어진다.

CBL 연결:

| 표어 | CBL 단계 |
|---|---|
| 좋은 질문 | Essential Question |
| 탐구 | Investigation |
| 문제 해결 | Solution |
| 도전과 실천 | Challenge / Action |

## 6. “왜 굳이 방정식으로 표현해야 하는가?”

핵심 답변:

> 그림은 위치를 보여주는 것이고, 방정식은 위치의 조건을 판단하고 검증하게 해주는 도구이다.

### 요점

1. 그림은 대략적이지만, 방정식은 정확한 판단 기준을 제공한다.
2. 방정식은 조건이 바뀌어도 다시 분석할 수 있다.
3. 방정식은 보이지 않는 지점, 즉 조건을 만족하는 모든 점을 찾게 해준다.
4. 방정식은 주장을 수학적으로 설득하는 근거가 된다.
5. AI 시대에는 AI의 제안을 검증하고 판단하는 도구가 된다.

수업에서 쓸 수 있는 정리 문장:

> 도형의 방정식은 그림을 수식으로 바꾸는 기술이 아니라, 공간의 조건을 정확히 판단하고 설명하기 위한 수학적 언어이다.

## 7. 수업 주제 후보

### 7.1 학교 안전지도

> 도형의 방정식으로 만드는 우리 학교 안전지도

### 7.2 학교 안전시설물 위치 제안

> 우리 학교에서 안전 개선이 필요한 장소를 하나 선정하고, 도형의 방정식을 활용하여 안전시설물의 최적 위치를 제안하라.

### 7.3 통학로 위험지역 분석

> 등하굣길 위험지역을 좌표와 거리 조건으로 분석하고 개선안을 제안하라.

### 7.4 와이파이 음영지역 개선

> 학교 내 와이파이 신호가 약한 곳을 조사하고, 공유기 위치를 수학적으로 제안하라.

### 7.5 비상 대피 장소 또는 대피 경로 제안

> 여러 교실에서 안전하고 공평하게 접근 가능한 대피 장소나 경로를 제안하라.

## 8. 게임 맵 설계 논의

게임 맵 설계는 학생 흥미와 도형의 방정식 개념 연결은 좋음.

예시:

> 좌표평면 위에 두 팀의 기지, 아이템, 장애물, 안전구역을 배치하고, 도형의 방정식을 활용하여 공정한 게임 맵을 설계한 뒤 실제 테스트를 통해 개선하라.

그러나 CBL 철학에서 보면 한계가 있음.

핵심 논의:

- CBL의 Action은 단순 산출물 제작이 아니라 실제 세계와 연결된 영향력 있는 행동을 지향함.
- 게임 맵은 실제 사회나 학교 공동체 문제 해결과의 연결성이 약할 수 있음.
- 따라서 엄밀한 CBL로는 **학교 안전시설물, 통학로, 대피 경로, 와이파이 개선** 등이 더 적합함.

정리:

> 게임 맵 설계는 몰입도는 높지만 CBL의 사회적 실행성은 약하다. 학교 시설물·위험지역·안전지도 주제가 CBL 철학에 더 잘 맞는다.

## 9. 현재 가장 적절한 Big Idea

지도·학교 시설물·위험지역 수업의 Big Idea:

> 위치와 거리는 좌표와 방정식으로 표현할 수 있으며, 이를 통해 학교 공간의 안전 문제를 수학적으로 분석하고 개선할 수 있다.

더 간결한 표현:

> 수학은 공간의 안전과 효율을 판단하는 언어이다.

또는:

> 우리 학교의 위험지역과 안전시설물 위치는 어떤 수학적 기준으로 정할 수 있을까?

## 11. 추천 Challenge

가장 적절한 도전과제:

> 우리 학교의 위험지역을 수학적으로 분석하고, 안전시설물 설치 위치를 제안하여 학교 공동체에 공유하라.

수업용으로 구체화하면:

> 학교 지도 위에 위험지역과 주요 시설물을 좌표로 나타내고, 거리·직선·원의 방정식을 활용하여 안전시설물의 설치 위치를 수학적 근거와 함께 제안하라.

학생용 제시문:

> 여러분은 학교 안전 개선팀입니다. 학교 안에서 위험하거나 개선이 필요한 장소를 하나 찾고, 그 주변의 시설물과 이동 경로를 좌표평면 위에 나타내세요. 그리고 도형의 방정식을 활용하여 안전 표지판, CCTV, 비상벨 중 하나의 설치 위치를 제안하세요. 단, 제안 위치는 반드시 거리, 직선 또는 원의 방정식으로 설명해야 합니다.

## 12. 단원 범위를 넘어서는 탐구에 대한 입장

단원을 넘어서는 탐구는 CBL에서 자연스러운 현상이다.

가능한 확장:

- 최단거리
- 통계적 위험도 분석
- 실제 동선 데이터
- 확률적 사고
- 최적화
- 알고리즘
- 지도 축척
- 이동 시간

하지만 평가의 중심은 해당 단원의 성취기준에 두어야 한다.

핵심 원칙:

> 탐구는 열어두되, 평가는 단원 성취기준에 묶는다.

또는:

> 확장은 허용하되, 수학적 근거의 중심은 도형의 방정식에 둔다.

### 필수 개념

- 좌표평면
- 두 점 사이 거리
- 직선의 방정식
- 원의 방정식

### 확장 개념

- 최적화
- 통계
- 확률
- 실제 이동거리
- 가중치
- 데이터 분석
- 알고리즘
- 지도 축척

## 13. 평가 방향

평가는 다음 원칙으로 설계한다.

> 창의적인 확장은 인정하되, 평가의 중심은 도형의 방정식 개념을 활용한 수학적 설명에 둔다.

### 평가 비율 예시

| 평가 영역 | 비율 | 평가 내용 |
|---|---:|---|
| 도형의 방정식 개념 활용 | 40% | 좌표, 거리, 직선, 원의 방정식 활용 |
| 문제 정의와 수학적 모델링 | 25% | 위험지역 선정, 기준 설정, 수학적 조건화 |
| 해결안의 타당성 | 20% | 제안 위치의 근거, 현실 적용 가능성 |
| AI 활용과 성찰 | 15% | AI 사용 기록, 피드백 반영, 한계 성찰 |

### 루브릭 예시

| 평가 요소 | 우수 | 보통 | 미흡 |
|---|---|---|---|
| 도형의 방정식 활용 | 거리, 직선, 원의 방정식을 정확히 사용하고 의미를 설명함 | 일부 개념을 사용했으나 설명이 부족함 | 식이 없거나 단원 개념과 관련이 약함 |
| 문제 정의 | 학교 공간 문제를 구체적으로 정의하고 수학적 기준으로 바꿈 | 문제는 제시했으나 기준이 모호함 | 단순 장소 소개에 그침 |
| 수학적 근거 | 제안 위치가 계산, 식, 그래프와 논리적으로 연결됨 | 계산은 있으나 제안과 연결이 약함 | “좋아 보인다” 수준의 설명 |
| 협업과 의사소통 | 역할 분담, 토론 과정, 발표가 명확함 | 협업 흔적은 있으나 역할이 불분명함 | 개인 작업 중심이거나 소통 부족 |
| AI 활용 | AI를 질문·피드백 도구로 쓰고 사용 기록과 수정 과정을 제시함 | AI 사용 기록은 있으나 반영 과정이 약함 | AI 결과를 그대로 사용하거나 기록 없음 |
| 성찰 | 모델의 한계와 개선 방향을 구체적으로 설명함 | 한계 인식은 있으나 일반적임 | 성찰이 형식적임 |

## 14. 2차시 지도안 초안

### 수업 주제

> 도형의 방정식으로 만드는 우리 학교 안전지도

### 수업 개요

| 항목 | 내용 |
|---|---|
| 과목 | 고등학교 수학 |
| 단원 | 도형의 방정식 |
| 수업 주제 | 학교 공간의 위험지역을 좌표와 방정식으로 분석하고 개선 위치 제안하기 |
| 수업 형태 | CBL 기반 탐구·문제 해결 수업 |
| 핵심 개념 | 좌표평면, 두 점 사이 거리, 직선의 방정식, 원의 방정식 |
| AI 활용 | 질문 생성, 기준 설정, 수학적 설명 피드백, 발표문 개선 |
| 최종 산출물 | 우리 학교 안전지도 + 수학적 근거 + 개선 제안서 |

### 수업 목표

학생은 수업 후 다음을 할 수 있다.

1. 학교 지도 위의 시설물과 위험지역을 좌표평면에 나타낼 수 있다.
2. 두 점 사이 거리, 직선의 방정식, 원의 방정식을 활용하여 공간 문제를 분석할 수 있다.
3. 위험지역 또는 안전구역을 수학적 조건으로 표현할 수 있다.
4. AI를 활용하여 탐구 질문을 생성하고, 자신의 수학적 설명을 점검할 수 있다.
5. 수학적 근거를 바탕으로 학교 공간 개선안을 제안할 수 있다.

## 17. 최종 산출물 구조

```md
1. 우리 조가 정한 문제
- 예: 급식실 앞 통로가 좁고 학생 이동량이 많아 위험하다.

2. 좌표 설정
- 급식실 입구 A(3, 5)
- 계단 B(7, 5)
- 차량 출입구 C(5, 1)

3. 사용한 수학 개념
- 두 점 사이 거리
- 직선의 방정식
- 원의 방정식

4. 수학적 분석
- 위험지역 A를 중심으로 반경 3 이내를 주의 구역으로 설정
- (x-3)² + (y-5)² ≤ 9
- 주요 이동 경로는 y = 5로 표현 가능

5. 개선 제안
- 안전 표지판을 P(5, 5)에 설치한다.
- 이유: 급식실과 계단 사이 이동 경로 위에 있으며, 위험 반경과 겹치는 지점이다.

6. AI 활용 기록
- 사용 목적: 기준 설정 피드백, 발표문 점검
- 사용 프롬프트:
  “우리 조의 안전지도 분석에서 수학적 근거가 부족한 부분을 질문으로 피드백해줘.”
- AI 결과 중 반영한 점:
  “위험 반경을 왜 3으로 정했는지 설명해야 한다는 피드백을 반영함.”

7. 성찰
- 방정식으로 표현하니 위험지역의 범위를 더 명확하게 판단할 수 있었다.
- 실제 이동 경로는 직선거리가 아니므로 한계가 있다.
```

## 20. 최종 핵심 문장 모음

> AI 시대에는 답을 아는 것보다, 어떤 문제를 정의할 것인가가 더 중요하다.

> CBL은 학생이 문제를 푸는 사람이 아니라, 문제를 정의하고 도전하는 사람으로 성장하게 하는 수업 설계 방식이다.

> 좋은 질문은 탐구를 만들고, 탐구는 문제 해결을 만들며, 문제 해결은 도전과 실천으로 이어진다.

> 그림은 위치를 보여주지만, 방정식은 위치의 조건을 설명한다.

> 도형의 방정식은 단순한 계산 단원이 아니라, 공간의 문제를 수학적으로 모델링하고 해결하는 언어이다.

> AI는 정답을 대신 내는 도구가 아니라, 학생이 더 좋은 질문을 만들고 자신의 판단을 점검하도록 돕는 도구로 활용된다.

> 탐구는 열어두되, 평가는 단원 성취기준에 묶는다.

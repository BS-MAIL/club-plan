# 대푯값 수업 웹앱

학생들이 각자 태블릿으로 접속해 자료를 입력하고, 교사용 화면에서 평균/중앙값/최빈값을 실시간으로 확인하는 로컬 웹앱입니다.

## 실행 방법

```bash
npm install
npm start
```

교사용 화면: `http://localhost:3000/teacher`

학생용 화면: `http://localhost:3000/student`

학생 태블릿은 교사 컴퓨터와 같은 와이파이에 연결한 뒤, 서버 실행 화면에 표시되는 `http://교사PC_IP:3000/student` 주소로 접속합니다.

## 기능

- 교사가 여러 숫자 주제를 추가합니다.
- 학생은 이름, 조, 주제별 값을 제출합니다.
- 평균, 중앙값, 최빈값을 자동 계산합니다.
- 자료의 분포에 따라 적절한 대푯값을 자동 추천하고, 교사가 평균/중앙값/최빈값으로 바꿀 수 있습니다.
- 랜덤 주제를 뽑아 조별 대표 학생과 설명을 제출합니다.
- 정답 공개 시 가장 가까운 조에 1점을 부여합니다.

## Railway 배포 메모

나중에 Railway에 올릴 때도 `npm start`로 실행하면 됩니다. Railway가 제공하는 `PORT` 환경 변수를 자동 사용합니다.

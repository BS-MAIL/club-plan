# 배로 불어나는 세계: 등비수열 실험실

`kkh0405/README.md`의 1차 MVP 범위를 바탕으로 만든 정적 웹앱입니다.

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 실행됩니다.

## 포함 기능

- 초항, 공비, 항의 개수, 실생활 시나리오 조작
- 수열 표 자동 생성
- SVG 막대그래프와 점/선그래프
- 일반항 카드와 스토리 결과 문장
- 예측값 입력, 정답·오차·힌트 확인
- 미션 4개와 달성 여부 표시
- 탐구 질문 제공
- 개인정보 없는 브라우저 로컬 탐구 로그
- JSON/CSV 로그 저장
- 큰 글씨 모드와 결과 복사

## 파일 구성

```text
app/
├── index.html
├── styles.css
├── app.js
└── README.md
```

# kkh0405 - 등비수열 인터랙티브 웹앱 수업 기획

> **주제**: 등비수열을 이용한 재미있는 실습형 웹앱 수업 설계  
> **핵심 방향**: 학생이 공비와 초항을 직접 조작하며 등비수열의 증가·감소·폭발적 변화·실생활 적용을 발견하는 웹 기반 탐구 수업  
> **연계 산출물**: MAIL 동아리 산출물 A, B, C 중 `산출물 A: 2차시 수업 지도안`의 기획 초안

---

## 1. 수업 앱 콘셉트

### 1.1 앱 가칭

**「배로 불어나는 세계: 등비수열 실험실」**

### 1.2 한 줄 설명

학생들이 웹앱에서 초항, 공비, 항의 개수, 상황 시나리오를 조작하며 등비수열의 규칙과 일반항을 스스로 발견하고, 실생활 문제를 해결하는 탐구형 수학 실험실입니다.

### 1.3 핵심 재미 요소

- **즉각적인 시각 변화**: 공비를 바꾸면 막대그래프, 점그래프, 표, 스토리 결과가 바로 변합니다.
- **게임형 미션**: “10번째 날에 목표 금액을 넘겨라”, “바이러스 확산을 1000명 이하로 막아라” 같은 도전 과제를 제공합니다.
- **예측 후 확인**: 학생이 먼저 다음 항이나 특정 항을 예측하고, 앱이 실제 결과와 오차를 보여줍니다.
- **실생활 시나리오**: 용돈 불리기, 세포 분열, 조회수 증가, 감염 확산, 반감기, 할인율 등 친숙한 맥락을 사용합니다.
- **탐구 로그**: 학생이 바꾼 값과 예측 결과를 기록하여 수업 후 과정 중심 평가 자료로 활용합니다.

## 2. 교육적 문제 상황

등비수열은 공식 `a_n = a_1 r^{n-1}`을 외우는 방식으로 끝나기 쉽습니다. 학생들은 공비가 수열의 변화 양상을 어떻게 결정하는지, 왜 지수적으로 증가하거나 감소하는지, 공비가 1보다 클 때와 0과 1 사이일 때 어떤 차이가 있는지 직관적으로 이해하지 못하는 경우가 많습니다.

이 웹앱은 학생이 직접 값을 조작하고 결과를 관찰하게 하여 “공식 암기”보다 “규칙 발견”을 먼저 경험하게 하는 것을 목표로 합니다.

## 4. 웹앱 주요 기능 기획

### 4.1 메인 화면: 등비수열 실험실

학생은 다음 값을 슬라이더 또는 입력창으로 조절합니다.

| 조작 변수 | 설명 | 예시 |
|:---|:---|:---|
| 초항 `a₁` | 처음 값 | 1, 2, 1000 |
| 공비 `r` | 매번 곱해지는 수 | 0.5, 1.2, 2, -1 |
| 항의 개수 `n` | 관찰할 항 개수 | 5~20 |
| 시나리오 | 실생활 맥락 | 용돈, 세포, 조회수, 할인 |

화면에는 다음 요소가 동시에 표시됩니다.

1. **수열 표**: `n`, `a_n`, 이전 항과의 관계
2. **막대그래프**: 항별 크기 비교
3. **점그래프/선그래프**: 변화 추세 확인
4. **일반항 카드**: `a_n = a_1 × r^(n-1)` 형태로 자동 생성
5. **스토리 결과**: “10일 후 값은 512입니다”처럼 맥락화된 문장

### 4.2 미션 모드

학생이 목표를 달성하도록 초항과 공비를 조절하는 모드입니다.

#### 미션 예시

| 미션 | 목표 | 수학 개념 |
|:---|:---|:---|
| 용돈 두 배 챌린지 | 10일 뒤 10만 원 이상 만들기 | 공비가 1보다 큰 등비수열 |
| 할인 마스터 | 5번 할인 후 가격 예측하기 | 0<r<1인 등비수열 |
| 세포 분열 실험 | 특정 시간 후 세포 수 예측 | 지수적 증가 |
| 소문 확산 막기 | 7단계 후 1000명 이하로 제한 | 공비 조절과 예측 |
| 반감기 탐구 | 일정 시간마다 절반이 되는 양 찾기 | 공비 1/2 |

### 4.3 예측 모드

학생은 앱이 결과를 보여주기 전에 특정 항을 예측합니다.

1. 앱이 `a₁`, `r`을 제시합니다.
2. 학생이 `a_5`, `a_8`, `a_10` 등을 입력합니다.
3. 앱이 정답, 오차, 풀이 힌트를 보여줍니다.
4. 학생은 “내가 사용한 규칙”을 짧게 기록합니다.

### 4.4 탐구 질문 생성 기능

앱 또는 교사용 자료에 다음 질문을 제공합니다.

- 공비가 2일 때와 3일 때, 10번째 항은 얼마나 차이가 나는가?
- 공비가 1보다 크면 그래프는 어떤 모양이 되는가?
- 공비가 0과 1 사이이면 값은 어떻게 변하는가?
- 공비가 음수이면 항의 부호와 그래프는 어떻게 달라지는가?
- 초항이 달라도 공비가 같으면 변화의 “느낌”은 어떻게 비슷한가?
- 같은 목표값에 도달하는 서로 다른 초항과 공비 조합을 만들 수 있는가?

### 4.5 학습 데이터 로그 후보

개인정보 없이 다음 데이터를 수집할 수 있습니다.

| 데이터 | 활용 목적 |
|:---|:---|
| 학생이 선택한 미션 | 흥미 있는 맥락 파악 |
| 조작한 초항·공비 값 | 탐구 과정 분석 |
| 예측값과 실제값의 차이 | 개념 이해도 확인 |
| 힌트 사용 여부 | 어려움 정도 파악 |
| 완료한 미션 수 | 참여도 확인 |
| 자가 평가 응답 | 정의적 반응 분석 |

> 실제 구현 시 학생 이름, 학번 등 개인정보는 저장하지 않고 익명 코드 또는 기기별 임시 ID를 사용합니다.

| 탐구 과정 | 여러 값을 체계적으로 비교하고 근거를 기록함 | 일부 값을 비교하고 결과를 기록함 | 무작위 조작에 그치고 기록이 부족함 |
| 문제 해결 | 미션 조건을 수학적으로 분석하고 적절한 전략을 세움 | 시행착오를 통해 목표에 도달함 | 목표 조건을 식으로 연결하지 못함 |
| 디지털 도구 활용 | 그래프·표·로그를 근거로 설명함 | 도구 결과를 확인하고 활용함 | 도구 조작은 하나 결과 해석이 부족함 |
| 협업·소통 | 역할을 나누고 팀의 전략을 명확히 발표함 | 팀 활동에 참여하고 결과를 공유함 | 참여가 제한적이거나 설명이 부족함 |

### 9.3 학생 자가 평가 문항

- 나는 등비수열에서 공비의 의미를 설명할 수 있다. `1~5점`
- 나는 등비수열의 일반항을 이용해 특정 항을 구할 수 있다. `1~5점`
- 웹앱 조작이 등비수열을 이해하는 데 도움이 되었다. `1~5점`
- 오늘 활동은 재미있었다. `1~5점`
- 가장 기억에 남는 실험 또는 미션은 무엇인가?
- 아직 헷갈리는 부분은 무엇인가?

# AI 활용 및 출처 검증 기록 초안

> **작성일**: 2026-07-06

---

## 1. AI 활용 목적

| 활용 단계 | AI 활용 목적 | 교사 검토 사항 |
|:---|:---|:---|
| 수업 설계 | CBL 주제 구체화, 차시 흐름 아이디어 정리 | 교육과정 및 성취기준과의 연결성 확인 |
| 학생 탐구 | 트래픽 사례 탐색, 그래프 모델링 피드백 | AI 응답의 수학적 타당성 확인 |
| 평가 | 학생별 AI 사용 로그를 수행평가 근거로 활용 | 로그 수집 범위와 개인정보 보호 기준 확인 |
| 세특 | 학생 활동 자료를 바탕으로 세특 초안 생성 보조 | 교사가 최종 검토 및 수정 |

---

## 2. 학생용 AI 사용 구상

- 학생은 교사가 제공한 링크를 통해 수업 시간에만 AI를 사용한다.
- AI 사용 과정은 자동 로그로 저장한다.
- 로그에는 질문, 응답, 재질문, 학생의 최종 반영 여부를 남긴다.
- AI가 제시한 내용은 반드시 학생이 수학적으로 검토하고 수정한다.
- AI 활용 여부 자체보다 비판적 검토와 수학적 설명 능력을 평가한다.

---

## 3. 출처 검증 계획

| 검증 대상 | 검증 방법 | 기록 방식 |
|:---|:---|:---|
| 트래픽 관련 실제 데이터 | 공공데이터, 기관 자료, 기사 등 원출처 확인 | 자료명, URL, 접근일 기록 |
| AI가 제시한 설명 | 교과서, 수업 자료, 신뢰 가능한 웹 자료와 대조 | 맞는 내용/수정한 내용 구분 |
| 그래프 모델 | 주기, 진폭, 평행이동 값이 데이터와 맞는지 확인 | 계산 과정과 그래프 캡처 저장 |
| 위험 시나리오 | 데이터 해석에서 과장이나 비약이 없는지 확인 | 판단 근거와 한계 기록 |

## 4. 추가 검토 필요 사항

- 공용 API 키 사용 가능 여부와 비용
- 수업 시간 외 링크 접근 제한 방법
- 학생별 로그 저장 시 개인정보 처리 기준
- 로그를 세특 및 수행평가 자료로 활용할 때의 안내 문구
- AI 응답의 출처 표기 및 검증 방식

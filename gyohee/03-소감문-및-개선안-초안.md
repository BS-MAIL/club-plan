## 1. 수업 설계 소감 초안

### 1.1 주제 선택 계기

삼각함수 그래프는 학생들이 식과 그래프의 형태를 암기하는 데 그치기 쉬운 단원이다. 그러나 교통량, 맛집 대기 줄, 놀이공원 이용객 밀집처럼 실제 생활에서 반복적으로 나타나는 트래픽 패턴을 연결하면 주기, 진폭, 최댓값과 최솟값, 평행이동의 의미를 더 자연스럽게 이해할 수 있다고 보았다. 학생들이 생활 속 데이터를 바탕으로 위험 상황을 예측하고 대안을 제시하도록 설계하면, 삼각함수가 현실 문제 해결에 쓰이는 도구라는 점을 경험할 수 있을 것이다.

### 1.2 AI 도구 선택 이유

AI는 학생이 선택한 트래픽 사례를 구체화하고, 데이터 해석 방향을 점검하며, 그래프 모델링 과정에서 피드백을 제공하는 보조 도구로 활용할 수 있다. 특히 학생들이 질문을 던지고 AI의 답변을 다시 검토하는 과정은 단순 정답 확인보다 탐구 과정을 드러내는 데 도움이 된다. 다만 AI 결과를 그대로 사용하는 것이 아니라, 학생이 수학적 근거를 들어 수정하고 검증하도록 안내할 필요가 있다.

## 2. 현장 적용 전 기대

### 2.1 수업 전 기대

학생들이 삼각함수 그래프를 공식 중심으로만 보지 않고, 반복되는 생활 현상을 설명하는 모델로 이해하기를 기대한다. 또한 AI 사용 로그를 통해 학생의 질문 방식, 탐구 과정, 피드백 반영 정도를 확인하면 수행평가와 세특 기록의 근거를 더 구체적으로 확보할 수 있을 것으로 기대한다.

### 2.2 예상되는 어려움

| 어려움 | 예상 해결 방법 |
|:---|:---|
| 실제 데이터를 구하기 어려움 | 교사가 예시 데이터 또는 간단한 시뮬레이션 데이터를 제공 |
| AI가 부정확한 식이나 해석을 제시함 | 학생이 주기·진폭·평행이동 의미를 직접 검증하도록 체크리스트 제공 |
| 수업 시간 외 AI 사용으로 평가 공정성 문제 발생 | 수업 시간에만 접근 가능한 링크와 로그 저장 방식 검토 |
| API 키와 사용량 관리 문제 | 공용 API 키 사용 가능 여부와 비용, 접근 제한 방식 사전 검토 |

- 학생용 AI 링크는 수업 시간에만 열리도록 설계하는 방향을 검토한다.

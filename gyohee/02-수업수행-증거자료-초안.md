## 3. AI 로그 수집 구상

### 3.1 자동 저장하고 싶은 항목

| 데이터 항목 | 수집 목적 | 비고 |
|:---|:---|:---|
| 접속 시간 | 수업 시간 내 AI 사용 여부 확인 | 수업 시간 외 접근 제한 검토 |
| 학생 질문 | 탐구 방향과 사고 과정 파악 | 개인정보 입력 금지 안내 필요 |
| AI 응답 | AI가 제공한 도움의 성격 파악 | 출처 검증 대상 |
| 학생 수정 내용 | AI 결과를 비판적으로 검토했는지 확인 | 수행평가 근거 |
| 최종 산출물 연결 | 로그와 결과물의 관계 확인 | 세특 작성 자료 |

### 3.2 평가 활용 아이디어

- AI를 많이 사용했는지보다, AI 응답을 검토하고 수학적으로 타당하게 수정했는지를 평가한다.
- AI가 제안한 그래프식이나 해석을 그대로 제출하지 않고, 학생이 직접 주기·진폭·평행이동의 의미를 설명했는지 확인한다.
- 로그는 학생의 탐구 과정, 질문의 깊이, 피드백 반영 정도를 보여 주는 보조 자료로 활용한다.

## 4. 개인정보 및 공정성 고려

- 학생은 이름, 전화번호, 정확한 위치 정보 등 개인정보를 AI에 입력하지 않는다.
- 공용 API 키 또는 학생용 링크를 사용할 경우 접근 권한과 사용 시간을 제한한다.
- 수업 시간 외 접속 제한 방식을 검토하여 수행평가의 공정성을 확보한다.
- 사진·영상 자료는 학생 얼굴과 개인정보를 가린 뒤 저장한다.

## 3. 주제 1: 직접 잴 수 없는 높이 구하기

### 3.1 수업 주제

삼각비로 우리 학교의 직접 잴 수 없는 높이를 구하라.

### 3.2 수업 목표

- 삼각비를 이용하여 직접 측정하기 어려운 높이를 구할 수 있다.
- 실제 측정값을 바탕으로 직각삼각형 상황을 모델링할 수 있다.
- 측정 오차의 원인을 분석하고 더 정확한 측정 방법을 제안할 수 있다.

### 3.3 핵심 질문

직접 올라가거나 재지 않고도 학교 안의 높은 물체의 높이를 어떻게 구할 수 있을까?

### 3.4 도전 과제

학교 안에서 직접 높이를 잴 수 없는 대상 하나를 정하고, 삼각비를 이용해 그 높이를 구한 뒤, 측정 오차를 줄이는 방법까지 제안하라.

### 3.5 대상 예시

- 학교 건물
- 나무
- 깃대
- 체육관 천장 조명
- 강당 무대 장치
- 농구 골대 위 구조물

### 3.6 기본 수학 모델

```text
tan theta = 높이 차 / 관측 지점과 대상 사이의 거리

높이 차 = 거리 x tan theta

실제 높이 = 높이 차 + 관측자의 눈높이
```

### 3.7 교수-학습 과정

| 단계 | 활동 내용 | 교사 역할 | 학생 활동 |
|:---|:---|:---|:---|
| 도입 | 직접 잴 수 없는 높이 문제 제시 | 학교 사진, 건물 사진 제시 | 높이를 구하는 방법 토의 |
| 문제 정의 | 측정 대상 선정 | 안전한 측정 가능 장소 안내 | 모둠별 측정 대상 선택 |
| 개념 연결 | 삼각비와 높이 측정 원리 설명 | `tan theta` 활용 구조 설명 | 직각삼각형 그림 그리기 |
| 자료 수집 | 거리, 각도, 눈높이 측정 | 측정 도구 사용법 안내 | 줄자, 각도기 앱 등으로 측정 |
| 계산 | 삼각비로 높이 계산 | 계산 과정 피드백 | 높이 계산, 단위 정리 |
| 검증 | 오차 원인 분석 | 비교 측정 방법 안내 | 여러 지점에서 측정해 비교 |
| 발표 | 결과와 개선 방법 공유 | 발표 기준 제시 | 계산 과정, 결과, 오차 분석 발표 |

### 3.8 준비물

- 줄자 또는 거리 측정 도구
- 스마트폰 각도기 앱 또는 클리노미터 앱
- 계산기
- 활동지
- 학교 공간 사진 또는 지도

### 3.9 학생 활동지 구성

```text
1. 측정 대상:
2. 측정 위치:
3. 대상까지의 거리:
4. 관측 각도:
5. 관측자의 눈높이:
6. 사용한 삼각비:
7. 계산 과정:
8. 최종 높이:
9. 오차가 생길 수 있는 이유:
10. 더 정확하게 측정하기 위한 방법:
```

### 3.10 평가 기준

- 삼각비 선택이 적절한가?
- 직각삼각형 모델을 정확히 세웠는가?
- 계산 과정과 단위가 명확한가?
- 측정 오차를 합리적으로 분석했는가?
- 결과를 논리적으로 설명했는가?

### 3.11 확장 질문

- 측정 위치를 더 멀리 잡으면 결과가 더 정확해질까?
- 각도를 여러 번 측정하면 오차를 줄일 수 있을까?
- 같은 대상을 두 지점에서 측정하면 결과가 왜 달라질까?

## 4. 주제 2: 안전한 경사로 설계하기

### 4.1 수업 주제

삼각비로 안전하고 편리한 경사로를 설계하라.

### 4.2 수업 목표

- 삼각비를 이용하여 경사로의 길이, 높이, 경사각을 구할 수 있다.
- 경사로의 기울기와 안전성 사이의 관계를 설명할 수 있다.
- 주어진 조건에 맞는 경사로 설계안을 만들 수 있다.

### 4.3 핵심 질문

누구나 안전하게 이용할 수 있는 경사로는 어떤 각도와 길이를 가져야 할까?

### 4.4 도전 과제

학교 안의 계단이나 단차가 있는 공간을 하나 선택하고, 휠체어, 유모차, 카트가 안전하게 이동할 수 있는 경사로를 설계하라.

### 4.5 대상 예시

- 교실 입구의 단차
- 건물 현관 계단
- 강당 무대
- 급식실 입구
- 도서관 입구
- 운동장과 건물 사이의 단차

### 4.6 기본 수학 모델

```text
높이 = h
바닥 거리 = d
경사로 길이 = l
경사각 = theta

tan theta = h / d
sin theta = h / l
cos theta = d / l
```

### 4.7 교수-학습 과정

| 단계 | 활동 내용 | 교사 역할 | 학생 활동 |
|:---|:---|:---|:---|
| 도입 | 학교 안 단차 사진 제시 | 이동 약자의 관점 질문 | 불편하거나 위험한 공간 찾기 |
| 문제 정의 | 개선이 필요한 공간 선정 | 안전한 조사 범위 안내 | 모둠별 대상 장소 선택 |
| 개념 연결 | 경사로와 직각삼각형 연결 | 높이, 바닥 거리, 경사로 길이 설명 | 경사로 단면도 그리기 |
| 조건 설정 | 경사각 또는 길이 조건 정하기 | 현실적 조건 제시 | 안전한 경사로 기준 정하기 |
| 계산 | 삼각비로 필요한 길이 계산 | 계산 피드백 | 경사로 길이, 바닥 거리, 각도 계산 |
| 설계 | 경사로 도면 제작 | 설계 요소 안내 | 단면도, 평면도, 설명 작성 |
| 발표 | 설계안 발표 및 비교 | 평가 기준 안내 | 안전성, 공간 활용성 설명 |

### 4.8 준비물

- 줄자
- 계산기
- 각도기
- 활동지
- 학교 공간 사진
- 모눈종이 또는 디지털 도면 도구

### 4.9 학생 활동지 구성

```text
1. 개선할 장소:
2. 현재 문제점:
3. 단차의 높이:
4. 사용할 수 있는 바닥 거리:
5. 목표 경사각:
6. 필요한 경사로 길이:
7. 사용한 삼각비:
8. 설계 도면:
9. 이 설계가 안전한 이유:
10. 현실적으로 설치할 때 고려할 점:
```

### 4.10 평가 기준

- 경사로 상황을 직각삼각형으로 잘 표현했는가?
- 높이, 바닥 거리, 경사로 길이의 관계를 삼각비로 설명했는가?
- 설계 조건이 현실적인가?
- 안전성과 편의성을 함께 고려했는가?
- 수학적 계산이 설계와 연결되어 있는가?

### 4.11 확장 질문

- 경사각이 작아지면 경사로 길이는 어떻게 변할까?
- 공간이 부족하면 어떤 대안을 생각할 수 있을까?
- 안전한 경사로와 짧은 경사로 중 무엇을 우선해야 할까?

## 5. 주제 3: 최적의 위치 선정하기

### 5.1 수업 주제

삼각비로 가장 좋은 위치를 찾아라.

### 5.2 수업 목표

- 삼각비를 이용하여 특정 대상을 바라보거나 비추는 데 필요한 각도와 거리를 구할 수 있다.
- 위치, 높이, 각도 사이의 관계를 수학적으로 설명할 수 있다.
- 목적에 맞는 최적의 설치 위치나 관측 위치를 제안할 수 있다.

### 5.3 핵심 질문

어떤 대상을 가장 잘 보이게 하거나 정확히 비추려면 어디에 위치해야 할까?

### 5.4 도전 과제

학교 안의 특정 대상이 잘 보이거나 잘 비추어지도록 카메라, 조명, 안내판, 현수막, 관측 지점 중 하나의 최적 위치를 정하라.

### 5.5 가능한 세부 주제

- 운동장 행사를 촬영하기 좋은 카메라 위치 찾기
- 무대 발표자를 잘 비추는 조명 위치 정하기
- 현수막이 가장 잘 보이는 설치 높이와 거리 정하기
- 복도 안내판이 잘 보이는 위치 정하기
- 체육대회 사진 촬영을 위한 관측 지점 정하기
- 농구 골대나 무대의 특정 지점을 바라보는 최적 위치 찾기

### 5.6 기본 수학 모델

```text
대상과 관측자의 높이 차 = h
관측자와 대상 사이의 수평 거리 = d
시선 거리 = l
관측각 = theta

tan theta = h / d
sin theta = h / l
cos theta = d / l
```

### 5.7 교수-학습 과정

| 단계 | 활동 내용 | 교사 역할 | 학생 활동 |
|:---|:---|:---|:---|
| 도입 | 잘 보이는 위치와 안 보이는 위치 비교 | 사진 또는 상황 제시 | 위치에 따른 차이 토의 |
| 문제 정의 | 최적 위치가 필요한 상황 선정 | 선택 가능한 주제 제시 | 모둠별 문제 상황 선택 |
| 조건 설정 | 좋은 위치의 기준 정하기 | 기준 예시 제공 | 잘 보임, 안전, 거리, 각도 기준 설정 |
| 개념 연결 | 시선, 각도, 거리의 관계 설명 | 직각삼각형 모델 제시 | 상황을 도식화 |
| 측정 | 높이 차와 거리 측정 | 측정 방법 안내 | 실제 공간에서 자료 수집 |
| 계산 | 삼각비로 각도 또는 거리 계산 | 계산 점검 | 후보 위치별 값 비교 |
| 의사결정 | 최적 위치 선정 | 근거 중심 발표 안내 | 수학적 근거로 위치 선택 |
| 발표 | 최종 위치 제안 | 피드백 | 도면, 계산, 이유 발표 |

### 5.8 준비물

- 줄자
- 스마트폰 각도기 앱
- 계산기
- 학교 지도 또는 공간 사진
- 활동지
- 모눈종이

### 5.9 학생 활동지 구성

```text

3. 후보 위치 1:
4. 후보 위치 2:
5. 후보 위치 3:
6. 각 위치에서의 거리:
7. 높이 차:
8. 관측각 또는 비추는 각도:
```

### 5.10 평가 기준

- 현실적 조건을 함께 고려했는가?
- 발표에서 선택 이유가 설득력 있는가?

### 5.11 확장 질문

- 가장 가까운 위치가 항상 가장 좋은 위치일까?
- 각도가 너무 크거나 작으면 어떤 문제가 생길까?
- 수학적 최적 위치와 실제 설치 가능한 위치가 다르면 어떻게 해야 할까?

| 2차시 | 학교 공간 개선 미션 | 경사로 또는 최적 위치 중 선택하여 설계 | 간단한 개선 제안서 |

## 7. 평가 계획

### 7.1 과정 중심 평가

| 평가 요소 | 평가 내용 | 평가 방법 |
|:---|:---|:---|
| 문제 이해 | 학교 공간 문제를 명확히 정의했는가 | 활동지, 관찰 |
| 수학적 모델링 | 실제 상황을 직각삼각형으로 표현했는가 | 도식, 계산 과정 |
| 삼각비 활용 | 상황에 맞는 삼각비를 선택하고 계산했는가 | 활동지, 발표 |
| 협업 | 역할을 나누고 측정과 분석에 참여했는가 | 관찰, 동료 평가 |
| 검증과 개선 | 오차 원인을 분석하고 개선 방법을 제안했는가 | 보고서, 발표 |

### 7.2 루브릭 초안

| 수준 | 설명 |
|:---|:---|
| 상 | 실제 상황을 적절히 모델링하고, 삼각비 계산과 오차 분석을 근거로 설득력 있는 해결안을 제시한다. |
| 중 | 삼각비를 이용해 필요한 값을 구하고, 해결안을 대체로 타당하게 설명한다. |
| 하 | 삼각비 계산이나 상황 모델링에 일부 어려움이 있으나, 교사의 도움을 받아 기본 문제를 해결한다. |

## 8. 교사용 유의점

- 학생들이 측정할 수 있는 공간을 사전에 확인하고, 위험한 장소는 제외한다.
- 스마트폰 각도기 앱을 사용할 경우 기기별 오차가 있을 수 있음을 안내한다.
- 삼각비 계산보다 실제 상황을 직각삼각형으로 해석하는 과정에 충분한 시간을 둔다.
- 경사로 설계 활동에서는 실제 법적 기준을 엄밀히 적용하기보다, 중3 수준에서 기울기와 안전성의 관계를 이해하는 데 초점을 둔다.
- 최적 위치 선정 활동에서는 학생들이 스스로 기준을 정하고 비교하도록 유도한다.

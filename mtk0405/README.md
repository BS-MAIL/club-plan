# mtk0405: 직선과 원의 방정식 인터랙티브 수업 앱 기획

> **주제**: 직선의 방정식과 원의 방정식을 이해하는 학생 참여형 웹 실습 앱 설계  
> **대상**: 고등학교 수학 학습자  
> **핵심 방향**: 학생이 직선과 원의 그래프를 직접 움직이고 식의 변화를 관찰하며, 방정식의 각 요소가 그래프에 어떤 의미를 가지는지 이해하는 웹 기반 수학 실험실

## 2. 프로젝트 한 줄 설명

**학생들이 좌표평면 위에서 직선과 원을 직접 조작하며, 직선의 기울기·절편과 원의 중심·반지름이 방정식에 어떻게 나타나는지 발견하는 인터랙티브 웹앱**을 만든다.

---

## 3. 핵심 교육 질문

1. 왜 직선의 방정식은 기울기와 한 점만으로 결정될까?
2. 원의 방정식에서 중심과 반지름은 식의 어느 부분에 숨어 있을까?
3. 직선의 기울기와 y절편을 바꾸면 그래프는 어떻게 움직일까?
4. 원의 중심과 반지름을 바꾸면 식과 그래프는 어떻게 달라질까?
5. 학생은 그래프를 보고 직선과 원의 방정식을 추론할 수 있을까?
